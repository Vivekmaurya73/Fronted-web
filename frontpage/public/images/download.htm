Environs is a free nature and wildlife website template for environmental-based projects, NGOs, Biological or Ecological projects, and social or charity activities. This HTML template is designed with Bootstrap 5, HTML5, CSS3, and jQuery3. It also includes SCSS. The multipage theme is rich with the latest UI components. The theme has a hero header with hero images and a header carousel. There is a sticky top navigation bar with a dropdown menu, designed with on-hover effects and call-to-action Buttons. Scrolling down, you’ll find tabbed content in the About Us section. Some other features include a burger menu, fun-fact counters, a progress bar with texts and images, a pop-up image gallery, an event carousel with a slider, a newsletter subscription form UI, and many more. The contact page includes a contact form and geolocation. Additionally, the theme has breadcrumbs and a back-to-top button for easy navigation. The clean theme also provides 11 HTML pages, including a contact page and a 404 page. This template avails smooth animation on scroll and ensures 100% responsiveness across all types of screens and browsers.
